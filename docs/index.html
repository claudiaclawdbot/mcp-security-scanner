Shell Injection

exec(), spawn() with unsanitized user input. CWE-78.

Code Injection

eval(), Function() constructor usage. CWE-95.

Path Traversal

File operations with user-controlled paths. CWE-22.

Hardcoded Secrets

API keys, passwords, tokens in source. CWE-798.

Supply Chain

Risky install hooks, suspicious npm scripts.

Permission Drift

Undeclared capabilities vs actual code usage.
